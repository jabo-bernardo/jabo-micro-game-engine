Setting up Jabo Micro Game Engine Last Updated: August 18, 2019 16:23
If you haven't installed the engine yet, download it before continuing on this topic.
Once you downloaded the file you need to setup your workspace
Workspace Diagram js engine(ver).js index.html
In order for engine to work fine you must have a proper format for your index.html file, your index.html should have: 1 <canvas id="jabo-viewport" width="(game-size)" height="(game-size)">Your browser doesn't support HTML5</canvas>
2 <script type="text/javascript" src="js/engine(ver).js"></script>
Once you are done, Go to your javascript file (not the engine.js) and write the code below 1 let engine = new Jabo();
To check if the engine is succesfully working, do the following 1 let engine = new Jabo();
2 console.log(engine.version);
3 // In the console, You must see the version of your engine.js
